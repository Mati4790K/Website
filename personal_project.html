Personal Project

Last year I started doing a computer game. This is a tower defense strategy game, where user has to use different towers and strategies to win the game. I really liked the game and I keep working on it.

Since September I added a lot of different features to my game e.g. I added main menu for the game, where player can visit options menu, credits menu, exit the game or play the game. Firt photo from the left shows the main menu, second photo shows options menu, third photo shows credits menu, and last photo shows game over screen when I lost all my lives while playing the game.


I use Unity game engine (first photo from left) to create my game and I create all 3D assets by myself using Blender 3D editing software (second photo from left).
